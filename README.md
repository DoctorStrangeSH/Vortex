# 🌪️ Vortex Messenger

Современный мессенджер с открытым исходным кодом.

## 🚀 Особенности
- 🔒 End-to-end шифрование
- 🤖 AI-помощник
- 🌍 Автоперевод
- 📦 Работает оффлайн
- 💾 Экспорт/импорт данных

## 🛠 Технологии
- Bootstrap 5
- Firebase (Auth + Firestore)
- Vanilla JavaScript

## 🏃 Быстрый старт

2. Открой index.html в браузере
3. Готово!
